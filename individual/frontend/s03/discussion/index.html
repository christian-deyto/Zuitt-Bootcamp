HTML Images

An image can be shown using <img> tag along its src attribute.

<img src="https://place-puppy.com/300x300">

<img src="./images/cat.jpg">

Valid image extensions are the following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp

Take a look at w3schools attribute references here.
